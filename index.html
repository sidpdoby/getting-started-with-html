  • HTML HyperText Markup Language

    HTML is the heart of the WEB and is composed of:

    • Text, which is the content (what you see).
    • Markup, what the content looks like.
    • Referenes to other documents, i.e. images and videos.
    • Link to oter pages.

    • URLS which provide references to documentss
    • HTTP, the protocol of the web or the transport medium
    • Web applications

  • Tags and Elements

    Tags are primarily composed of opening and closing tags. However, there are tags called 'void' which do not require a closing tag.

    Failing to properly close a tag may result in everything following the opening tag to assume that attribute of that tag. For example, if one were to use an opening tag of "em" and fail to close that tag, everything from the opening tag forward would be shown in italics.

    Sample tags include:

    • b, for bolding text between opening and closing tags
    • em, for italicizing text
    • br, creates a break between text.
    • p, paragraph, does just that. LOL. It has content.

    Attributes are properties of HTML and used properly, create powerful pages.
    Attributes include:

    • HREF, which creates a link to access additional information/site. For example: Udacity
    • IMG, which does not contain a closing tag. It is considered a "void" tag. In the example below the use of 'alt' creates an alternate view of the referenced source. This is quite helpful should a web error occur and supports a blind computer user.'

  • Inline vs Block Elements

    Inline elements include:

    • a, used to create links such as derp Opening and closing tags used.
    • br, used toens a line and subsequently wraps text to next line. No closing tag.
    • em, emphasizes and italicizes words. Opening and closing tags used.
    • img, introduces a picture. No closing tag
    • span, uses opening and closing tags
    • strong, uses opening and closing tags

    Block elemeents include:

    • p, which creates in invisible "block" around everything included within opening and closing tags. Additional attributes such as
      height and width may also be applied.
    • form and its closing tags where developer is creating a form for a user to complete.

      • Addtionally these block elements may be considered 'container' elements and have content.

      • div and its closing tag. Syntax::
        text
      • span and its closing tag. Syntax: text , where ‘class’ is a CSS style
